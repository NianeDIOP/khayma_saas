Middleware ResolveTenant

Résout le tenant depuis le sous-domaine (prod) ou ?_tenant=slug (local). Injecte la company dans le conteneur IoC.

En cours
En production : {slug}.khayma.com. En développement local : localhost/app?_tenant=monrestaurant. Les sous-domaines réservés (www, admin, api, staging, mail, smtp) sont rejetés.
  • Créer app/Http/Middleware/ResolveTenant.php — méthode handle() + extractSlug() dev
  • Résolution depuis ?_tenant=slug sur localhost / 127.0.0.1 dev
  • Résolution depuis premier segment du sous-domaine en production — rejeter les réservés dev
  • Injecter dans IoC : App::instance('currentCompany', $company) dev
  • Partager avec Inertia : inertia()->share('currentCompany', [...]) dev
  • Enregistrer l'alias 'tenant' dans bootstrap/app.php cfg 
// app/Http/Middleware/ResolveTenant.php
public function handle(Request $request, Closure $next): Response
{
    $slug = $this->extractSlug($request);
    if (! $slug) abort(404);

    $company = Company::where('slug', $slug)->where('is_active', true)->first();
    if (! $company) abort(404);

    App::instance('currentCompany', $company);
    inertia()->share('currentCompany', ['id' => $company->id, ...]);

    return $next($request);
}

// Statuts subscription_status
