Middleware IsAdmin + alias 'admin'

Protéger toutes les routes /admin — aborte 403 si l'utilisateur n'est pas super admin.

Validé
  • Créer app/Http/Middleware/IsAdmin.php — vérifie Auth::check() et Auth::user()->is_super_admin, sinon abort(403) dev
  • Enregistrer l'alias 'admin' => \App\Http\Middleware\IsAdmin::class dans bootstrap/app.php cfg 
// app/Http/Middleware/IsAdmin.php
public function handle(Request $request, Closure $next): Response
{
    if (! Auth::check() || ! Auth::user()->is_super_admin) {
        abort(403, 'Accès réservé aux super admins.');
    }
    return $next($request);
}
